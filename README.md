# Scaffold smoke test

Static SaaS landing page generated by **PostHog founder mode**.

## Stack

- Plain `index.html` + `styles.css` + `script.js` — no build step.
- Tailwind CSS via CDN, so any utility class works.
- Inter via Google Fonts.
- Brand palette wired into `--brand-primary` / `--brand-accent` CSS variables.

## Sections

Per-section purpose-built layouts: Hero, Social proof, Features (3-up icon cards), How it works (numbered steps), Problem statement, Pricing tiers, FAQ accordion, Final CTA gradient band, Footer. Markdown in the spec's `copy_hooks` is parsed into structured chunks before rendering — bullets become cards, **bold** lines followed by paragraphs become FAQ Q&A pairs, etc.

## Enabling analytics

Add to `<head>`:

```html
<meta name="posthog-key" content="phc_xxxxxxxxxxxxxxxxxxx">
```

## Iterating

Edit `index.html` and `styles.css` directly. Commit + push — GitHub Pages auto-deploys.
